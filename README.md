# 改进型人工势场法 (Improved APF) 无人机三维避障仿真系统

## 📖 项目简介

本项目是一个基于 MATLAB/Simulink 的单旋翼/多旋翼无人机三维路径规划与控制仿真平台。项目核心复现了 **Pan 等人提出的改进型人工势场法 (Improved Artificial Potential Field)**。

传统的人工势场法 (APF) 虽然计算简单、实时性高，但在实际应用中存在两个致命的数学缺陷：

1. **目标不可达问题 (GNRON, Goals Non-Reachable with Obstacles Nearby)**：当目标点附近有障碍物时，极大的斥力会导致无人机永远无法到达终点。
2. **局部极小值死锁 (Local Minima Deadlock)**：当无人机、障碍物、目标点处于同一直线时，引力与斥力大小相等方向相反，导致无人机在障碍物前停滞不前或陷入高频震荡。

本项目通过引入**带目标距离惩罚的修正斥力场**与**防死锁切向旋转力**，在物理仿真层面上完美攻克了上述两大难题，实现了无人机在复杂三维环境中的平滑绕障与精准到达。

## ✨ 核心特性 (Features)

* **解决 GNRON 问题**：在斥力函数中引入了无人机到目标点距离的 $n$ 次方项。当无人机靠近目标点时，即使旁边有障碍物，斥力也会随着目标距离的减小而衰减至零，确保 100% 精准触达终点。
* **破除死锁的切向切入机制**：实时监测引力与斥力的夹角（$\cos\theta$）。当面临正面对冲（夹角接近 180 度）死锁风险时，自动激活一个与斥力梯度正交的“切向旋转力 ($F_e$)”，引导无人机从侧面丝滑绕过障碍物。
* **全三维空间动力学仿真**：非简单的二维质点运动学规划，而是结合了无人机三维空间的刚体非线性动力学模型进行闭环控制。
* **高质量 3D 轨迹渲染**：配备专业的可视化脚本，一键生成带有 3D 障碍物球体光影渲染、无人机安全警戒圈以及动态飞行过程的动画演示。

## 📂 文件结构 (Project Structure)

* `init_APF.m`（或类似名称）：全局初始化脚本。用于配置无人机的质量/重力等物理参数、起点与终点坐标、障碍物位置与大小，以及 APF 的关键增益（$\eta$, $k_{att}$, $n$ 等）。
* `UAV_APF_Sim.slx`（或您的 Simulink 模型名称）：Simulink 仿真主框架，包含无人机动力学对象 (Plant) 和控制器模块。
* `APF_Controller.m`：核心控制算法模块。负责计算常规目标引力 ($F_{att}$)、改进型障碍物斥力 ($F_{rep}$) 以及防死锁旋转力 ($F_e$)。
* `plot_APF.m`（或类似名称）：仿真结果数据提取与 3D 可视化绘图脚本。

## 🚀 快速开始 (Quick Start)

### 1. 环境要求

* MATLAB / Simulink (建议 R2020a 或更高版本)

### 2. 运行步骤

**步骤一：初始化参数**
在 MATLAB 命令行中运行初始化脚本，加载所有物理与控制参数到工作区：

```matlab

>> init_APF
```

**步骤二：运行 Simulink 仿真**
打开 Simulink 模型文件，点击 **Run (运行)**。求解器将根据引入的势场合力，解算无人机从起点到终点的三维运动状态。

**步骤三：渲染可视化结果**
仿真跑完后，在命令行运行绘图脚本：

```matlab

>> plot_APF
```

系统将自动弹出 Figure 窗口，展示包含障碍物实体、警戒圈边界以及无人机平滑绕障过程的 3D 视图。

## ⚙️ 参数调试指南 (Tuning Guide)

如果你想测试不同的避障极限场景，可以通过修改 `init` 脚本中的参数来观察效果：

* **引斥力平衡**：`k_att`（引力增益）与 `eta`（斥力增益）。如果无人机躲避障碍物时显得太迟钝，可适当提高 `eta` 或增大警戒半径 `obs_rho0`。
* **GNRON 惩罚指数**：`n_goal`（通常设为 1 或 2）。该值决定了靠近终点时斥力衰减的速度。
* **速度阻尼**：`k_d`。在 APF 控制中，合理的线性速度阻尼能够有效吸收无人机进入/离开势场时的弹簧震荡能量，防止轨迹出现多余的 S 形波纹。
* **死锁测试**：可尝试将起点、障碍物中心、终点放置在严格的同一直线上（例如 Y 和 Z 坐标完全相同），验证防死锁切向旋转力（增益 `k_e`）能否成功将无人机“推离”死锁线。
